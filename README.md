> Aplicação web para gerenciamento e arquivamento de documentos Word, construída com **ASP.NET MVC** e **C#**.

---

## ✨ Funcionalidades

- 📁 Upload e armazenamento de documentos `.docx`
- 📋 Listagem e organização de arquivos
- 🔍 Busca e filtragem de documentos
- 🗑️ Exclusão e gerenciamento de arquivos
- 🏗️ Arquitetura MVC (Model-View-Controller)
- ✅ CRUD completo de documentos

---

## ⚙️ Tecnologias

| Tecnologia | Uso |

---

## 🚀 Como rodar localmente

```bash
# 1. Clone o repositório
git clone https://github.com/AlanPrates/Arquivo-Word-ASP-NET-MVC-.git
cd Arquivo-Word-ASP-NET-MVC-

# 2. Restaure os pacotes
dotnet restore

# 3. Rode a aplicação
dotnet run

# 4. Acesse no navegador
# http://localhost:5000
```

**Pré-requisito:** .NET 6+ instalado
